# StadiaFlow AI 🏟️

**StadiaFlow AI** is a modern, interactive crowd management platform designed to optimize the physical event experience at large-scale sporting venues, specifically cricket stadiums. It provides stadium managers with a high-fidelity command center to monitor crowd density, coordinate security, and optimize fan flow in real-time.

---

## 🚀 The Problem
Large-scale sporting events often face significant challenges:
- **Crowd Congestion**: Bottlenecks during ingress and egress.
- **Long Waiting Times**: Frustrating queues at concessions and restrooms.
- **Security Coordination**: Difficulty in pinpointing incident locations in massive crowds.
- **Real-time Communication**: Delayed response to changing physical dynamics (e.g., innings breaks).

## 💡 The Solution
StadiaFlow AI addresses these challenges through an integrated management dashboard that visualizes stadium dynamics using AI-powered analytics and interactive mapping.

### Key Features:
- **Symmetrical Stadium Engine**: A custom-built SVG engine that maintains a perfectly balanced layout of the stadium stands and facilities.
- **Dynamic Heatmaps**: Visualizes crowd density across sectors (Green/Yellow/Red status).
- **Tabbed Operational Views**:
    - **Dashboard**: High-level KPIs (Occupancy, Avg. Flow).
    - **Heatmaps**: Deep dive into density indices.
    - **Concessions**: Real-time stall monitoring and queue length tracking.
    - **Security**: Live CCTV placement and SOS incident markers.
    - **Crowd Flow**: Directional indicators to guide fans through the most efficient paths.
- **Simulation Modes**: Predictive modes for **Match Start**, **Innings Break**, and **Post-Match Exit** to prepare staff for surges.

## 🛠️ Technology Stack
- **Frontend**: HTML5, Vanilla JavaScript (ES6+)
- **Styling**: Modern CSS3 (Sporty aesthetic, Glassmorphism elements)
- **Mapping**: Dynamic SVG path generation for sub-pixel precision and layering.
